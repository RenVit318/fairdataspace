Manual Endpoints

You can also add SPARQL endpoints manually. These will be queried alongside the auto-discovered endpoints. Enter one endpoint per line in the format: URL | Label

Format: endpoint_url | label (one per line). The label is optional.

Security Recommendation

The dashboard uses credentials from the server environment variables (DASHBOARD_SPARQL_USERNAME / DASHBOARD_SPARQL_PASSWORD) to authenticate with the SPARQL endpoints.

Best practice: Create a dedicated dashboard-reader role on each AllegroGraph instance with the following restrictions:

  • Read-only access (no SPARQL UPDATE permissions)
  • Limited to specific repositories or named graphs
  • Restricted to aggregate queries only where possible
